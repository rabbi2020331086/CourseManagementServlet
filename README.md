# Course Management System (Java Servlet)

A full-stack web application for managing academic courses, built with Java Servlets and JSP as part of university coursework.

## Features

- CRUD operations for course management (Create, Read, Update, Delete)
- Server-side rendering with JSP templates
- MVC architecture pattern
- HTTP request handling with Java Servlets
- Session management and form validation

## Tech Stack

| Technology | Purpose |
|-----------|---------|
| Java | Backend logic |
| Servlet API | HTTP request handling |
| JSP | Server-side rendering |
| HTML/CSS | Frontend UI |
| Apache Tomcat | Application server |

## Architecture

The project follows the MVC (Model-View-Controller) pattern:
- **Model**: Java classes representing course entities
- **View**: JSP pages for user interface
- **Controller**: Servlet classes handling HTTP requests

## Getting Started

### Prerequisites
- Java JDK 8+
- Apache Tomcat 9+
- IDE with Java EE support (e.g., IntelliJ IDEA, Eclipse)

### Setup
1. Clone the repository
2. Import as a Java EE project in your IDE
3. Configure Apache Tomcat server
4. Deploy and run
